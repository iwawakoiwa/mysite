## やりたいこと

ESP32 + MPU9250 で自作 SlimeVR トラッカーを作る。  
VRChat でフルボディトラッキングしたい。

## SlimeVR とは

オープンソースのフルボディトラッキングシステム。  
IMU センサーを体の各部位に装着して姿勢推定する。

## 進捗

まだ設計段階。MPU9250 の I2C 通信を Rust で書いているところ。

```rust
use esp_idf_hal::i2c::*;

// MPU9250 のデバイスアドレス
const MPU9250_ADDR: u8 = 0x68;
```

ファームウェアが安定したら基板設計に入る予定。
